# Security Criticals — Fix in Dedicated Session

Generated by `/cso` audit on 2026-03-26. These need dedicated attention.

## CRITICAL (Fix First)

### 1. Unauthenticated JWT Issuance
- **File:** `src/routes/auth.js`
- **Issue:** signup/login issue JWT without verifying credentials against any user store
- **Impact:** Complete account takeover — anyone can get a JWT for any userId
- **Fix:** Verify credentials against Supabase Auth before issuing tokens. Require email/phone verification for signup.

### 2. WhatsApp Webhook No HMAC (**DELETED** — no longer applies)
- WhatsApp route has been removed from the project entirely.

## HIGH (Fix After Criticals)

### 3. Fyers OAuth State Bypass
- **File:** `src/routes/fyers.js:74-81`
- **Issue:** Falls back to `req.query.userId` when state validation fails
- **Impact:** Attacker can link their Fyers account to any victim's userId
- **Fix:** Enforce signed state token, remove query param fallback entirely

### 4. Rate Limiter Uses Untrusted userId
- **File:** `src/middleware/rateLimit.js`
- **Issue:** Rate limit keyed on JWT userId which is freely issuable (depends on #1)
- **Fix:** Fix #1 first. Add IP-based rate limiting as secondary layer.

### 5. Mass Assignment in Onboarding
- **File:** `src/routes/onboarding.js`
- **Issue:** `req.body` passed directly to repository without field allowlist
- **Fix:** Destructure and allowlist only expected fields

### 6. LLM Prompt Injection via User Facts
- **File:** `src/routes/chat.js` + `src/lib/ai.js`
- **Issue:** User facts interpolated into system prompt
- **Fix:** Move user context to user-role messages, not system prompt

### 7. Tool Dispatch Without Output Validation
- **File:** `src/tools/executor.js`
- **Issue:** Raw tool output returned to model without sanitization
- **Fix:** Sanitize tool outputs with delimiters

### 8. No Per-User AI Cost Cap
- **File:** `src/routes/chat.js`
- **Issue:** Tool loop has no iteration cap
- **Fix:** Cap at 5 tool calls per message, track daily usage

### 9. Upstox SDK Singleton Race Condition
- **File:** `src/brokers/index.js:93-94`
- **Issue:** Global `ApiClient.instance` token overwritten per request
- **Fix:** Create new ApiClient instance per request

## MEDIUM

### 10. Google Drive Query Injection
- **File:** `src/routes/drive.js`
- **Issue:** User search query interpolated into Drive API query without escaping
- **Fix:** Escape quotes and backslashes in user input
